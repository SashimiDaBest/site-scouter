# Frontend

Map-first React + Vite frontend for Renewables Site Scout.

## Commands

```bash
npm ci
npm run dev
npm run lint
npm run lint:fix
npm run format:check
npm run format
npm run test
npm run build
npm run preview
```

## Architecture Notes

- `src/App.jsx`: app state, region selection logic, validation, analysis workflow.
- `src/App.css`: map/background layering and UI overlay styling.
- `src/test/setupTests.js`: Testing Library matcher setup.
- `src/App.test.jsx`: UI/UX behavior tests.

## Deployment Notes

- CI and deployment workflows are in `../.github/workflows/`.
- GitHub Pages deployment publishes `dist` artifact generated by Vite build.
- Base path for Pages is injected in CI with `VITE_BASE_PATH`.
